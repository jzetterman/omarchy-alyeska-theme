# Alyeska

A dark Omarchy theme with a deep navy `#1e1e2e` background, soft `#e4e8fb` foreground, and a muted `#5e8a60` for comments. Built on a Catppuccin-style ANSI palette and accented with `#89b4fa`.

## Install

Install with `omarchy-theme-install`:

```bash
omarchy-theme-install https://github.com/jzetterman/omarchy-alyeska-theme.git
```

This clones the repo into `~/.config/omarchy/themes/alyeska` and activates it.

To switch to it later:

```bash
omarchy-theme-set alyeska
```

Or pick it from the menu with <kbd>Super</kbd> + <kbd>Ctrl</kbd> + <kbd>Shift</kbd> + <kbd>Space</kbd>.

## Manual install

If you'd rather not use `omarchy-theme-install`:

```bash
git clone https://github.com/jzetterman/omarchy-alyeska-theme.git ~/.config/omarchy/themes/alyeska
omarchy-theme-set alyeska
```

## What's themed

- Hyprland (`hyprland.conf`, `hyprlock.conf`)
- Waybar, Walker, Wofi, Mako, SwayOSD
- Alacritty, Kitty, Ghostty, Warp
- Neovim, Zed, Zellij, btop
- GTK, Chromium, Vencord
- A VS Code extension under `vscode-extension/`
